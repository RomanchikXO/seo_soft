# ShagTeamPro

Инструмент для оптимизации карточек в Яндекс.Поиске и Яндекс.Картах с веб‑интерфейсом.

## Быстрый запуск

После клонирования репозитория достаточно одной команды — всё остальное (виртуальное
окружение, зависимости, браузер Chromium для Playwright) приложение настроит само
при первом запуске.

### Требования

- Установленный **Python 3.12 или новее** ([скачать](https://www.python.org/downloads/)).
  - На Windows при установке отметьте галочку **«Add Python to PATH»**.

### Шаг 1. Запуск

**Windows** — дважды кликните по файлу `run.bat`
(или в терминале: `run.bat`).

**macOS / Linux** — в терминале из папки проекта:

```bash
./run.sh
```

(если не запускается, сначала выдайте права: `chmod +x run.sh`)

**Любая ОС, вручную:**

```bash
python main.py
```

### Шаг 2. Работа

При первом запуске несколько минут уйдёт на установку зависимостей и Chromium —
это происходит один раз. После этого автоматически откроется браузер с интерфейсом
по адресу вида `http://127.0.0.1:<порт>`.

Чтобы остановить приложение — закройте окно терминала или нажмите `Ctrl + C`.

## Полезное

- Все зависимости ставятся в локальную папку `.runtime-venv` внутри проекта и не
  затрагивают системный Python.
- Список зависимостей — в `requirements.txt`.
- Запуск тестов:

```bash
python -m pytest
```
